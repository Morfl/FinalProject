# FinalProject

## Текущая функциональность
-  Пополнение счета
-  Списание средств
-  Проверка баланса


## Endpoints
GET  /getBalance/{id} - счет клиента
POST  /putMoney/{id}/money - Пополнение счета 
POST  /takeMoney/{id}/money - Списание средств
